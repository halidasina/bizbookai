# BizBook AI — Landing Page

Static landing page for BizBook AI Telegram bot.

## Deploy

### Netlify (recommended)
1. Go to [netlify.com](https://netlify.com) and sign up free
2. New site → Import from GitHub → select this repo
3. Build command: *(leave empty)*
4. Publish directory: `.`
5. Click Deploy

### Manual (Netlify Drop)
Drag the entire folder to [netlify.com/drop](https://app.netlify.com/drop)

## Files
- `index.html` — main landing page
- `images/` — all screenshots and logo assets
